
```js
// Opérateur de comparaison simple
false == 'false'    // false
false == '0'        // true
   '' == '0'        // false
    0 == ''         // true
    0 == '0'        // true
 null == undefined  // true
```
```js
// Opérateur de comparaison strict
false === 'false'   // false
false === '0'       // false
   '' === '0'       // false
    0 === ''        // false
    0 === '0'       // false
 null === undefined // false
```
